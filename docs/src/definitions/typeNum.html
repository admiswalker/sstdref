    typeNum

    Abstract

    typeNum defines the ID for the SSTD internal process according to the C++ type. typeNum では C++ の型に応じた ID を SSTD の内部処理向けに定義します．

    Description

    Function name Description
    type2num() returns the ID according to the C++ type.
    C++ の型に応じた ID を返します．

    Usage

    • main.cpp 
      #include <sstd/sstd.hpp>

int main(){
    bool tmp=true;
    sstd::printn(sstd::type2num(tmp));
    sstd::printn(sstd::type2num(bool()));
    sstd::printn(sstd::type2num(int64()));
    sstd::printn(sstd::type2num(std::vector<int64>()));
}
    • Execution result 
      sstd::type2num(tmp) = 0
sstd::type2num(bool()) = 0
sstd::type2num(int64()) = 6
sstd::type2num(std::vector<int64>()) = 20
